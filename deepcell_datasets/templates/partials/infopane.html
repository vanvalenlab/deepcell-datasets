    DeepCell Datasets is designed to store all relevant information for biological images. There are 3 types of data stores: 1) Experimental Data (Raw), 2) Annotation Data (Crowdsourcing), and 3) Training Data (Training). There are references at each stage that allow users to have a complete picture of any piece of data they wish to use to train deep learning models.

    While each field has validation to ensure information is entered correctly, the following provides a detailed description of what each field represents.
